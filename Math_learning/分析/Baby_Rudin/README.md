#### 紧集

- 定义：任意开覆盖有有限子覆盖
- 设 $K\subset Y\subset X$，则 $K$ 关于 $X$ 是紧的当且仅当 $K$ 关于 $Y$ 是紧的
- 度量空间的紧子集都是闭集
- 紧集的闭子集都是紧集
- $k$-方格是紧集
- 对于 $E\subset R^k$，下面三个命题等价：
  - $E$ 是闭且有界的
  - $E$ 是紧的
  - $E$ 的每个无限子集在 $E$ 内有极限点
- Weierstrass 定理：$R^k$ 中每个有界无限子集在 $R^k$ 中有极限点
